## 2. 사업 구조 (핵심 이해)

**외부 인식 vs 실제**:
```
외부 인식: "아이나비 블랙박스 회사"
실제 매출:  환경생활가전(로보락 유통) 66~70%
            차량용 블랙박스            ~30%
            지도/자율주행               ~3%
```

### 2-1. 환경생활가전 (매출 1위 사업)

- **로보락(Roborock) 국내 독점 유통** — 중국산 프리미엄 로봇청소기
- 공기청정기, 음식물처리기 등 생활가전 유통
- 매출 급증의 핵심이지만 **마진율이 낮음** (유통업 특성)
- 2025년 1Q 매출 +48%인데 영업이익 급감한 이유 = 로보락 판매 확대

### 2-2. 차량용 블랙박스 (아이나비 브랜드)

- **블랙박스 시장 점유율 1위** (~44%, 다나와 2023년 기준)
- **내비게이션 브랜드파워 22년 연속 1위** (2026년 퍼스트브랜드 대상)
- **BMW 80개국 OEM 공급** — 글로벌 확장 중
- 2024년 블랙박스 수출이 처음으로 내수를 초과
- 최근 제품: QXD2 (A.I. ISP, NPU 영상처리), QUANTUM3 (듀얼레이다 4K)

### 2-3. 지도/자율주행

- 자율주행 Level 2+/3용 통합지도 개발
- 완성차 OEM 공급 (현대/기아 등)
- 국책과제 수행 중

## 3. IT 인프라 환경 추론

**근거**: JD 내용 + 사업 규모 + 유사 중견기업 일반 패턴

### 예상 인프라 환경

| 영역 | 추정 환경 | 근거 |
|------|---------|------|
| **사내 네트워크** | 방화벽, L2/L3/L4 스위치 구성 | JD에 명시 |
| **서버** | Linux(CentOS/Ubuntu) + Windows Server 혼용 | JD에 명시 |
| **클라우드** | AWS 일부 사용 추정 | JD에 "Public Cloud(AWS)" 명시 |
| **Private Cloud** | 온프레미스 VM 환경 추정 | JD에 "Private Cloud" 명시 |
| **보안장비** | UTM/방화벽, IPS/IDS | JD에 "보안장비" 명시 |
| **스토리지** | NAS/SAN 구성 가능성 | JD에 "스토리지 구축/관리" 명시 |
| **인트라넷** | ERP, 그룹웨어, 사내 포털 | JD에 "사내 인트라넷 시스템 관리" 명시 |

### IT 인프라팀이 지원하는 시스템 (추론)

```
1. 사내 업무 시스템
   - ERP (물류, 재고, 재무) — 로보락 유통 규모가 크므로 중요
   - 그룹웨어 (이메일, 협업툴)
   - 사내 포털/인트라넷

2. 블랙박스 서비스 연동
   - 아이나비 앱 연동 서버 (원격 영상, 클라우드 백업)
   - OTA 업데이트 서버
   - 펌웨어 배포 인프라

3. 지도 서비스 인프라
   - 지도 데이터 업데이트 서버
   - 완성차 OEM 연결 네트워크

4. 글로벌 연결
   - 북미/일본 해외 법인 WAN/VPN
   - BMW OEM 파트너 연결 네트워크

5. 개인정보 처리 시스템
   - 차량 위치 데이터, 고객 정보
   - 개인정보보호법 대응 보안 체계
```

### IT팀 규모 추정

```
260명 기업의 IT 인프라팀 일반 규모: 5~15명
신입 채용 중 → 기존 팀원이 있고 보조/교육 역할로 진입
"넓게 하지만 깊이는 얕은" 환경일 가능성
→ 특정 기술을 깊게 파기보다 전반적 인프라 운영 경험 쌓기에 좋음
```

## 4. 복지/문화 현실 분석

### 잡플래닛 평점 (671건 기준)

| 항목 | 점수 |
|------|------|
| 복지/급여 | 2.1 |
| 워라밸 | **3.1** (상대적으로 높음) |
| 사내문화 | 2.7 |
| 승진기회 | 2.6 |
| 경영진 | 2.2 |
| **추천율** | **25%** |
| 성장 가능성 | 8% |

### 현실적 해석

**좋은 것**:
- 워라밸 3.1 → "칼퇴는 가능한 편"이라는 후기 다수
- 판교라는 위치 → 업계 네트워크 형성에 유리
- 블랙박스 1위 브랜드 → 이력서에 쓸 수 있는 회사명
- 근속 5.9년 → 짧지 않은 편 (불만족하면 금방 떠나는 경향)

**주의할 것**:
- 추천율 25% / 성장 가능성 8% → 커리어 성장을 기대하기 어려울 수 있음
- 복지/급여 2.1 → JD에 적힌 복지가 실제와 다를 수 있음
- 로보락 유통 매출 의존 → 로보락과의 계약이 흔들리면 회사 체력에 영향
